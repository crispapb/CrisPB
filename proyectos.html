UN VIAJE DE POESIA, MISTERIO Y TERROR

Para este proyecto se realizaron distintos materiales multimedia, entre los entregables se encuentran:

- Cartel promocional (optimizado para redes sociales)
- Cartel promocional (optimizado para impresión)
- Boleto en tamaño para impresión
- Programa de mano
- Fotografías del elenco y la puesta en escena

Todos estos elementos se realizaron completamente personalizados con las tipografías, logos y/o elementos necesarios e importantes para el grupo organizador que realizo la puesta en escena.

1. Todo se entrego para la facilidad de mi cliente, por ejemplo, se acordó que los programas de mano serían unicamente de 1 hoja doblada, de manera que con solo imprimir por las dos caras ya estaría listo para la distribución entre el público.

2. Los boletos tienen el mismo estilo y elementos que el cartel publicitario, pero a estos se les añadio marcas de corte, así una vez aprobado el diseño final, se entrego un documento tamaño tabloide con las marcas de corte para facilitar el corte de los mismos.

3. En el paquete personalizado, se acordó tener 2 sesiones fotográficas, una que serían fotografías "estandar" que servirian tanto para los acotres, así como eran necesarias para el contenido del programa de mano, y la segunda sesión sería en el ensayo general para que el grupo tuviera imagenes de stock para publicidad.

FOTOMONTAJE PROFESIONAL PARA DISEÑO DE CALZADO

En este proyecto, transformé la imagen base de un zapato, combinándola con diseños gráficos que se me proporcionaron en plastas de color, para crear un montaje mas realista para poder visualizar el diseño final. El resultado permitió presentar el producto de manera atractiva y profesional, ideal para presentaciones o catálogos. Los entregables acordados fueron:

- Montajes individuales de cada diseño (3 vistas c/u).
- Compilación completa en una sola imagen por diseño.

Todos los elementos fueron elaborados de manera completamente personalizada, incorporando los diseños, colores y detalles específicos necesarios para representar fielmente la esencia del calzado y cumplir con los requerimientos del cliente.

Se hizo un tratamiento a la Imagen base, para mejorar resolución, y los detalles de la textura, igualmente a cada imagen se le hcieron los siguientes ajustes de manera individual:

- Corrección de iluminación y detalles para lograr un acabado profesional.
- Fotomontaje del mapa de trabajo, integrando los diseños en la imagen base.
- Tratamiento de luces y texturas para fusionar de forma natural ambas imágenes.
- Ajuste del logotipo con una perspectiva estimada para una visualización realista.
- Tratamiento adicional para mejorar la definición general de la imagen, resaltando su calidad final.
